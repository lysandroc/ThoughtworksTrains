# Thoughtworks, Challenge Trains

### Make sure you have installed Dotnet Core SDK (v2.2 or latest).

# Quick-Start Guide


## Development Workflow

**Run build**
```bash
$ dotnet restore
```
```bash
$ dotnet build
```
**Run test**
```bash
$ dotnet test
```
**Run code**
```bash
$ dotnet dotnet run -p ThoughtworksTrains.Application/ThoughtworksTrains.Application.csproj`
```
